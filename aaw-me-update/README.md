# AAW.me Website Update Project

Project home for planning and executing updates to [www.aaw.me](https://www.aaw.me).

## Current Snapshot

- Brand: AAW, Mechanics of Sound / Advanced AcousticWerkes.
- Platform signal: Shopify storefront.
- Primary navigation: Home, Custom Builder, Get Started, Where to Buy, Gallery, Contact Us.
- Primary category entry points: For Musician, For Audiophile, TOTL, Wireless, Accessories.
- Homepage product focus observed on May 22, 2026: Z06 Universal In-Ear Monitor and Canary Pro Hybrid In-Ear Monitor.
- Key service paths: Universal Fit, Custom Fit, Reshell, Accessories.

## Project Goals

1. Improve the clarity of the buying paths for universal IEMs, custom IEMs, reshell services, and accessories.
2. Make the homepage and product/category journeys feel more premium, scannable, and conversion-oriented.
3. Preserve AAW's audiophile and musician credibility while making product selection easier for newer customers.
4. Identify operational updates needed in Shopify: theme, navigation, product data, content, imagery, SEO, and tracking.

## Working Assumptions

- The update will build on the existing Shopify site unless a platform migration is explicitly chosen later.
- Initial work should prioritize audit, information architecture, content strategy, and high-value page templates.
- Product photography, dealer information, and custom-builder details may need owner-supplied source materials.

## Immediate Next Steps

1. Complete a fuller current-site audit across homepage, product pages, collection pages, custom builder, dealer locator, gallery, and contact flow.
2. Define target audiences and buying journeys.
3. Draft a revised sitemap and navigation model.
4. Prioritize a phase-one implementation scope.
5. Decide whether this workspace will store only planning assets or also Shopify theme code.
